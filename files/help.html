Загрузка существующих изображений Поддерживаются форматы JPEG, PNG. Можно отправлять как фотографии, так и файлы. Но лучше файлы, так telegram не будет снижать их разрешение. Генерация изображений из текста Если отправить этому боту текст в формате markdown - на его основе будет создано png-изображение. По умолчанию будут белые буквы на прозрачном фоне. Внутри minecraft это выглядит достаточно хорошо. Стилем можно управлять. Для этого первую строку нужно начать с ! и использовать через запятую опции: * c - выравнивание по центру (по умолчанию - выравнивание по левому краю) * f24 - использовать 24 размер шрифта (в px) * m0 - убрать отступы между абзацами * w1024 - генерировать изобращение шириной 1024px (по умолчанию - 3 * 128 = 384px) * wb - использовать белый фон (и черный цвет текста) * b30 - использовать фон с прозрачностью 30% Несколько важных моментов: * Числа внутри опций можно изменять. * Если в сообщении все строки начинаются с # (т.е. являются заголовками), то автоматически применяется конфиг !f26,c,m0 (размер шрифта 26, выравнивание по центру, без отступов между абзацами). Если такое поведение не нужно - его можно отменить, использовав ! без опций на первой строке. * Есть поддержка рендеринга TeX-формул с помощью MathJax 2.7.7 * На рендеринг изображения требуется около 5 секунд. Причина в том, что нужен запас времени для работы MathJax. Редактирование сообщений Редактирование сообщений работает. Можно как заменять отправленные файлы/изображения, так и менять текст, на основе которого были сгенерированы изображения. Важно помнить, что в minecraft изменения отразятся только после перезагрузки изображения с сервера. Есть три способа сделать это: * Взять в руки палочку (stick) и ткнуть правой кнопкой мыши по изображению * Выполнить команду /i reload (перезагрузит все изображения) * Снять изображение и повесить его еще раз Получение изображений в Minecraft * Получить изображение можно командой /i 1234, где вместо 1234 нужно указать id загруженного изображения * По умолчанию размер изображения - 3 блока в ширину, размер в высоту определяется на основе соотношения сторон изображения. * Изменить размер можно взяв в руку изображение и использовав команду вида /i w 5 или /i h 5 (для установки размера в 5 блоков по ширине/высоте соответственно). * Снять изображение можно нажав shift + правый клик * Если просто кликнуть правой кнопкой по изображению - в чате появится информация о нем. При нажатии на сообщение откроется браузер со страницей изображения и метаинформацией.
